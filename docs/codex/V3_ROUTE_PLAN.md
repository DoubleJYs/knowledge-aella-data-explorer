# V3.0 Route and Responsibility Plan

## 1. 路由总览

| 路由 | 端 | 页面 | 主要用户 | 说明 |
|---|---|---|---|---|
| `/app` | 用户端 | 用户工作台 | 普通用户 / 科研人员 | 查看知识库概览、进入地图、上传资料、查看上传状态 |
| `/app/map` | 用户端 | 知识地图 | 普通用户 / 科研人员 | 只展示 published 知识节点 |
| `/app/search` | 用户端 | 搜索页 | 普通用户 / 科研人员 | 只搜索 published 条目 |
| `/app/upload` | 用户端 | 上传资料 | 普通用户 / 科研人员 | 上传后进入 pending_review |
| `/app/uploads` | 用户端 | 我的上传 | 普通用户 / 科研人员 | 查看自己的上传审核状态 |
| `/app/kbs` | 用户端 | 可浏览知识库 | 普通用户 / 科研人员 | 查看可访问知识库 |
| `/admin` | 后台端 | 管理首页 | 管理员 | 查看待审核、已发布、任务状态 |
| `/admin/review` | 后台端 | 审核队列 | 管理员 | 审核、发布、驳回 |
| `/admin/manual-entry` | 后台端 | 手动新增 | 管理员 | 直接新增知识条目 |
| `/admin/knowledge-bases` | 后台端 | 知识库管理 | 管理员 | 管理知识库 |
| `/admin/items` | 后台端 | 条目管理 | 管理员 | 管理已发布条目 |
| `/admin/tags` | 后台端 | 标签/聚类 | 管理员 | V3 可先占位 |
| `/admin/settings` | 后台端 | 系统设置 | 管理员 | AI 默认关闭 |
| `/legacy/*` | 实验视图 | 原 Aella | 开发者 / 管理员 | 保留原可视化能力 |

## 2. 状态流转

```text
用户上传
  -> pending_review
  -> 管理员审核
      -> rejected
      -> published
  -> published 才进入用户端地图和搜索
```

## 3. 用户端禁止能力

- 审核
- 发布
- 驳回
- 直接手动新增 published 条目
- AI Provider 设置
- 系统调试
- 地图重建任务控制

## 4. 后台端核心能力

- 审核队列
- 审核详情
- 通过并发布
- 驳回并填写原因
- 手动新增知识条目
- 管理知识库
- 查看导入 / 地图任务
- 系统设置
